# WorkAdventure — Render Deploy Setup

Este repositório contém a configuração mínima para rodar o WorkAdventure no Render usando Docker.

## Como usar

1. No Render clique em **New Web Service**
2. Escolha **Deploy via Docker**
3. Conecte este repositório
4. Adicione variável de ambiente:

```

DOMAIN=URL_DO_RENDER_OU_DOMINIO

```

5. Deploy
